## 프로젝트 구조

> 참고용이고, 나중에 유동적으로 변경 가능합니다

- `frontend/`: 브라우저 UI, 
- `backend/`: 게임 로직, 게임 방
- `shared/`: 프론트엔드-백엔드 간 사용하는 데이터 객체 등 저장
- `data/`: 로그 저장
- `assets/`: 이미지, 사운드, 폰트 등의 리소스
- and more!

## 역할분담

- 어승경 - 리눅스 서버 환경, websocket 구현
- 정시은 - 백엔드1 (게임 룸 생성, 유저 관리 등)
- 방주현 - 백엔드2 (게임 흐름 로직)
- 정연진 - 프론트엔드 (web UI)
- 염도윤 - 프론트엔드 및 백엔드 통합, 문서 작성 등

## 사용 기술
> 단순 저(어승경)가 생각한 적절한 기술 스택입니다. 원한다면 다른 기술 스택으로도 개발하셔도 돼요
- frontend: React
- backend: Node.js
- Storage: SQLite
- etc..

## MVP 목표
- 시민, 의사, 경찰, 마피아..? 정도의 기본적인 직업군 구현
- 진행되는 게임 방은 하나만 (MVP 구현 후 다중 방 생성 기능 추가, 이전에는 게임 진행 중일 때 새로 접속한 유저는 대기)
- 이외의 접속, 닉네임 입력, 대기방, 역할 배정, 밤/낮, 투표, 승리 판정 등 마피아 게임 로직
- websocket으로 실시간 게임 흐름 동기화
